================================================================================
  ARCHITECTURE PATTERN
================================================================================

  The app follows Angular's recommended feature-based folder structure:

  ┌─────────────────────────────────────────────────────┐
  │                    AppComponent                     │
  │                  (app.ts / app.html)                │
  └──────────────────────┬──────────────────────────────┘
                         │ Router Outlet
          ┌──────────────┼──────────────┐
          │              │              │
     HomeFeature    AuthFeature   StudentDashboard
          │
    ┌─────┴──────────────────────────────┐
    │  Hero  │  Features  │  HowItWorks  │  ...
    └────────────────────────────────────┘

  Shared Layer:  Navbar, Footer, HintBubble, DifficultyBadge
  Core Layer:    AuthService, ProblemService, ProgressService
  Models:        User, Problem, Progress, Hint
